# 📚 LibraryAPI - Sistema de Gestión de Biblioteca

Este es un sistema completo de gestión de bibliotecas que incluye un **Backend** robusto desarrollado con Node.js y SQLite, y un **Frontend** administrativo moderno.

## 🚀 Características
- **Gestión de Libros**: Registro, visualización y eliminación de inventario.
- **Control de Usuarios**: Sistema de registro y directorio de usuarios.
- **Préstamos**: Registro de préstamos con control automático de stock.
- **Base de Datos**: SQLite con modo WAL para mayor velocidad.

## 🛠️ Tecnologías utilizadas
- **Backend**: Node.js, Express, SQLite3, BcryptJS, JWT.
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (Fetch API).

## 📦 Instalación y uso
1. Clona el repositorio.
2. Ejecuta `npm install` para instalar las dependencias.
3. Crea un nuevo archivo`.env` basado en `.env.`.
4. Inicia el servidor con `npm run dev`.
5. Abre `http://localhost:3000/index.html` en tu navegador.
